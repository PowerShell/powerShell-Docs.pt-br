# Microsoft.PowerShell.Core Module

## Descrição

Esta seção contém os tópicos de ajuda para os cmdlets instalados com o módulo Microsoft. PowerShell. Core do PowerShell. O módulo principal contém cmdlets e provedores que gerenciam os recursos básicos do PowerShell.
