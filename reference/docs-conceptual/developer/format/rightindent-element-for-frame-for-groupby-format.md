# <a name="rightindent-element-for-frame-for-groupby-format"></a>Elemento RightIndent para Frame para GroupBy (formato)

Especifica quantos caracteres os dados são deslocados para fora da margem direita. Esse elemento é usado ao definir como um novo grupo de objetos é exibido.
