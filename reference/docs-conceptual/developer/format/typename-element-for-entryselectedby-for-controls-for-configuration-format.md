# <a name="typename-element-for-entryselectedby-for-controls-for-configuration-format"></a>Elemento TypeName para EntrySelectedBy para Controls para Configuration (formato)

Especifica um tipo .NET que usa essa definição do controle. Esse elemento é usado ao definir um controle comum que pode ser usado por todas as exibições no arquivo de formatação.
