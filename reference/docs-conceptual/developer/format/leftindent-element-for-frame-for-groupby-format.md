# <a name="leftindent-element-for-frame-for-groupby-format"></a>Elemento LeftIndent para Frame para GroupBy (formato)

Especifica quantos caracteres os dados são deslocados para fora da margem esquerda. Esse elemento é usado ao definir como um novo grupo de objetos é exibido.
