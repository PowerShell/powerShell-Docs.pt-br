# <a name="getproc02-code-samples"></a>Exemplos de código GetProc02

Aqui estão os exemplos de código para o cmdlet de exemplo GetProc02. Este é o exemplo de cmdlet `Get-Process` descrito em [adicionando parâmetros que processam a entrada de linha de comando](../cmdlet/adding-parameters-that-process-command-line-input.md). Esse cmdlet `Get-Process` recupera processos com base em seu nome e, em seguida, exibe informações sobre os processos na linha de comando.
