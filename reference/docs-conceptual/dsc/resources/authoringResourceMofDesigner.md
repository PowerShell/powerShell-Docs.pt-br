## <a name="creating-resource-properties"></a><span data-ttu-id="c5930-112">Criando propriedades de recurso</span><span class="sxs-lookup"><span data-stu-id="c5930-112">Creating resource properties</span></span>
<span data-ttu-id="c5930-113">A primeira coisa que precisamos fazer é decidir sobre as propriedades que serão expostas pelo recuso.</span><span class="sxs-lookup"><span data-stu-id="c5930-113">The first thing we need to do is decide on properties that the resource will expose.</span></span> <span data-ttu-id="c5930-114">Para esse exemplo, definiremos um usuário do Active Directory com as seguintes propriedades.</span><span class="sxs-lookup"><span data-stu-id="c5930-114">For this example, we will define an Active Directory user with the following properties.</span></span>

<span data-ttu-id="c5930-121">Para criar as propriedades, usamos o cmdlet **novo xDscResourceProperty**.</span><span class="sxs-lookup"><span data-stu-id="c5930-121">To create the properties, we use the **New-xDscResourceProperty** cmdlet.</span></span> <span data-ttu-id="c5930-122">Os seguintes comandos do PowerShell criam as propriedades descritas acima.</span><span class="sxs-lookup"><span data-stu-id="c5930-122">The following PowerShell commands create the properties described above.</span></span>

```powershell

$Ensure = New-xDscResourceProperty –Name Ensure -Type String -Attribute Write –ValidateSet "Present", "Absent"
$DomainCredential = New-xDscResourceProperty –Name DomainCredential -Type PSCredential -Attribute Write
$Password = New-xDscResourceProperty –Name Password -Type PSCredential -Attribute Write
```

## <a name="create-the-resource"></a><span data-ttu-id="c5930-123">Criar o recurso</span><span class="sxs-lookup"><span data-stu-id="c5930-123">Create the resource</span></span>

<span data-ttu-id="c5930-124">Agora que as propriedades do recurso foram criadas, podemos chamar o cmdlet **New-xDscResource** para criar o recurso.</span><span class="sxs-lookup"><span data-stu-id="c5930-124">Now that the resource properties have been created, we can call the **New-xDscResource** cmdlet to create the resource.</span></span> <span data-ttu-id="c5930-125">O cmdlet **New-xDscResource** utiliza a lista de propriedades como parâmetros.</span><span class="sxs-lookup"><span data-stu-id="c5930-125">The **New-xDscResource** cmdlet takes the list of properties as parameters.</span></span> <span data-ttu-id="c5930-126">Também usa o caminho no qual o módulo deve ser criado, o nome do novo recurso e o nome do módulo no qual ele está contido.</span><span class="sxs-lookup"><span data-stu-id="c5930-126">It also takes the path where the module should be created, the name of the new resource, and the name of the module in which it is contained.</span></span> <span data-ttu-id="c5930-127">O comando do PowerShell a seguir cria o recurso.</span><span class="sxs-lookup"><span data-stu-id="c5930-127">The following PowerShell command creates the resource.</span></span>

<span data-ttu-id="c5930-128">O cmdlet **New-xDscResource** cria o esquema MOF, um script de recurso esqueleto, a estrutura de diretório necessária para o novo recurso e um manifesto para o módulo que expõe o novo recurso.</span><span class="sxs-lookup"><span data-stu-id="c5930-128">The **New-xDscResource** cmdlet creates the MOF schema, a skeleton resource script, the required directory structure for your new resource, and a manifest for the module that exposes the new resource.</span></span>

<span data-ttu-id="c5930-129">O arquivo do esquema MOF está em **C:\Program Files\WindowsPowerShell\Modules\Demo_DSCModule\DSCResources\Demo_ADUser\Demo_ADUser.schema.mof** e seu conteúdo é o seguinte.</span><span class="sxs-lookup"><span data-stu-id="c5930-129">The MOF schema file is at **C:\Program Files\WindowsPowerShell\Modules\Demo_DSCModule\DSCResources\Demo_ADUser\Demo_ADUser.schema.mof**, and its contents are as follows.</span></span>

```

{
  [Key] string UserName;
  [Write, ValueMap{"Present","Absent"}, Values{"Present","Absent"}] string Ensure;
  [Write, EmbeddedInstance("MSFT_Credential")] String DomainCredential;
  [Write, EmbeddedInstance("MSFT_Credential")] String Password;
};
```

## <a name="updating-the-resource"></a><span data-ttu-id="c5930-133">Atualizando o recurso</span><span class="sxs-lookup"><span data-stu-id="c5930-133">Updating the resource</span></span>

<span data-ttu-id="c5930-137">Suponha, por exemplo, que você deseja incluir o horário do último logon para o usuário no nosso recurso.</span><span class="sxs-lookup"><span data-stu-id="c5930-137">For example, suppose you want to include the last log in time for the user in our resource.</span></span> <span data-ttu-id="c5930-138">Em vez de reescrever o recurso por completo, é possível chamar **New-xDscResourceProperty** para criar a nova propriedade e, depois, chamar **Update-xDscResource** e adicionar a nova propriedade à sua lista de propriedades.</span><span class="sxs-lookup"><span data-stu-id="c5930-138">Rather than writing the resource again completely, you can call the **New-xDscResourceProperty** to create the new property, and then call **Update-xDscResource** and add your new property to the properties list.</span></span>

```powershell
Update-xDscResource –Name 'Demo_ADUser' –Property $UserName, $Ensure, $DomainCredential, $Password, $lastLogon -Force
```
