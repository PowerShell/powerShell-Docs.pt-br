## <a name="my-resources-wont-update-how-to-reset-the-cache"></a><span data-ttu-id="cd478-202">Meus recursos não são atualizados: como redefinir o cache</span><span class="sxs-lookup"><span data-stu-id="cd478-202">My resources won't update: How to reset the cache</span></span>
