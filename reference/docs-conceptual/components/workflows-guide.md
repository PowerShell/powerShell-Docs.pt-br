# <a name="psworkflow-module"></a>Módulo PSWorkflow

Esta seção contém tópicos de ajuda para o módulo **PSWorkflow**, que inclui cmdlets que dão suporte ao recurso de fluxo de trabalho do Windows PowerShell. O módulo **PSWorkflow** está instalado com o Windows PowerShell 5.1.

> [!NOTE]
> Os fluxos de trabalho só têm suporte no Windows PowerShell 5.1.
